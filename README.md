# The Banana Bread Repository

A thoroughly tested banana bread implementation featuring brown butter, toasted pecans, and an irresponsible amount of chocolate.

A reproducible workflow for converting overripe bananas into happiness.

## Deployment

Upload `index.html`, `assets/`, and this `README.md` to the root of a public GitHub repository, then enable GitHub Pages from `main` and `/ (root)`.

The site includes an English / standard Mandarin toggle and all photos are stored in `assets/images/`.


## Features
- English / 普通话 language toggle
- Eight big recipe categories with detailed steps
- GitHub Pages ready
